Frequently Asked Questions

The tournament is open to all web designers and developers worldwide, regardless of experience level. You can participate as an individual or as a team (maximum of 3 members per team).

Participants will need to create a fully functional portfolio website. The technical requirements include:

  • Must be responsive and work on all device sizes
  • Should be deployed and publicly accessible
  • Can use any frontend technologies (HTML, CSS, JS, frameworks)
  • Backend is optional but can be included
  • Must follow the theme guidelines provided

Projects will be evaluated by our panel of expert judges based on the following criteria:

  • Design and aesthetics (40%)
  • User experience and functionality (30%)
  • Creativity and innovation (20%)
  • Code quality and technical implementation (10%)

Detailed judging rubric will be provided to all participants before the competition begins.
